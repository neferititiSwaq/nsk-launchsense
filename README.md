# 🚀 NSK LaunchSense: AI-Powered Go-To-Market Engine

**Empowering South African Entrepreneurs to Launch Smarter.**

NSK LaunchSense is an intelligent "Co-Founder" dashboard that transforms a simple product idea into a comprehensive launch strategy in seconds. Built for the **NSKdevpreneur Hub**, this tool automates market research, persona creation, and financial modeling.

---

## ✨ Key Features
* **👥 Persona Intelligence:** Generates hyper-local South African target personas (e.g., "The Weekend Warrior," "The Adventure Seeker").
* **🕵️ Competitor Mapping:** Scrapes real-time data to identify local market players and pricing benchmarks.
* **💰 Profit Architect:** A dynamic financial sidebar that calculates retail prices based on unit costs and desired margins.
* **🖼️ Visual Concepting:** Uses **Nano Banana 2 (Gemini 3 Flash Image)** to generate high-fidelity product sketches.
* **📄 Executive Export:** One-click PDF generation for a professional, stakeholder-ready business summary.

---

## 🛠️ The Tech Stack
* **Frontend:** [Streamlit](https://streamlit.io/) (Python-based Web UI)
* **AI Orchestration:** [Google GenAI SDK](https://ai.google.dev/) (Gemini 2.0 & 3.1 Flash)
* **Data Retrieval:** [Serper.dev](https://serper.dev/) (Google Search API)
* **Document Generation:** FPDF
* **Environment:** Python 3.12+
